# 🎥 视频二：拓展功能演示脚本 (Extended Requirements)

本视频重点演示环境温度监测系统的拓展与创新功能（总分值 40 分），展示系统对触摸屏的高级图形化应用、分页多传感器以及长时间数据图表展示的能力。

---

## 🎬 视频基本信息
*   **建议时长**：2 分钟 ~ 2.5 分钟
*   **展示功能**：触屏几何校准、双传感器分页、5 分钟实时曲线、触屏阈值设定 (1.0°C 步长)

---

## 📑 视频分幕脚本

### 🏁 第一幕：开机电阻屏交互式标定演示 (0:00 - 0:35)
*   **镜头画面**：
    *   演示者按住底板上的物理按键 **S0 (KEY_PAGE)** 并按下 Reset 复位键引导开机。
    *   LCD 屏幕进入白底的电阻屏校准引导页面，屏幕四个角依次出现红色十字靶心。
    *   演示者使用手写笔或手指依次点击屏幕上的 4 个靶心。
    *   校准成功后，屏幕亮起绿色文字 `Calibration Successful!` 并打印出 Raw 校准系数，随后系统载入主页。
*   **旁白配音**：
    *   “大家好，下面我将向大家演示系统的‘拓展与图形化交互功能’。首先展示电阻式触摸屏的交互式校准。在开机时按住 S0 按键引导，系统进入校准界面。通过手写笔点按四个角上的红色靶心，系统即可自动计算出极其精准的屏幕触摸映射矩阵。”
*   **字幕贴图**：
    *   ✅ 独立特色功能：开机按住 S0 自动引导演示
    *   ✅ 交互式 4 点电阻触屏标定与漂移消除

---

### 📄 第二幕：双传感器扩展与 LCD 触屏分页 (0:35 - 1:05)
*   **镜头画面**：
    *   镜头对准 Page 0（温湿页），随后演示者使用手指触摸屏幕右下角的切换区域（或者按下 **S0 物理按键**）。
    *   屏幕平滑无闪烁地切换到 Page 1（气压海拔页），实时显示气压 (Pa) 和高度值 (m)。
    *   再次点击触控区域，页面流畅切回 Page 0。
*   **旁白配音**：
    *   “本系统对传感器进行了拓展。除基础的 AHT20 外，我们还通过 I2C 总线接入了一路 BMP280 气压传感器。通过点按屏幕右下角的切换区域或者使用物理按键 S0，我们可以在‘温湿页’ and ‘气压海拔页’两个监测大屏之间进行流畅的分页显示与切换。”
*   **字幕贴图**：
    *   **拓展要求（1）多传感器扩展与分页**
    *   ✅ Page 0 (AHT20 温湿页) ⇄ Page 1 (BMP280 气压海拔页)

---

### 📈 第三幕：五分钟实时数据图表展示 (1:05 - 1:35)
*   **镜头画面**：
    *   微距拍摄 Page 0 中部的折线图区域。折线图以每秒 1 个点的速度向左推移，绘制当前温度的波动。
    *   演示者切换到 Page 1，折线图从红色温度线变为蓝色气压线，同样在向左移动。
*   **旁白配音**：
    *   “图表展示方面，本系统在中部设计了高精度、高灵敏度的实时动态折线图。我们在温湿页可以看到红色的温度走势线，在气压页可以看到蓝色的气压走势线。图表每秒更新一次，得益于我们开辟的 300 点大容量环形缓冲区，能连续无溢出地绘制并记录长达 5 分钟的环境历史波动趋势。”
*   **字幕贴图**：
    *   **拓展要求（2）图形化数据展示**
    *   ✅ 300点高容量缓冲区，直观折线波形
    *   ✅ 实时采样刷新（1 秒/点），记录 5 分钟历史趋势

---

### ⚙️ 第四幕：触摸屏图形化阈值设定 (1:35 - 2:15)
*   **镜头画面**：
    *   演示者点击主页面左下角的 **SETTINGS** 触控键（或按物理键 **S2 (KEY_BACK)**），屏幕切换为黑底的阈值设定面板。
    *   演示者点击温度上限 (Temp High) 的 **`UP / DOWN`** 虚拟触控按键，数值以 **1.0°C** 为整步单位增加或减少（例如从 29.0°C 调节到 32.0°C）。
    *   点击屏幕上的 **CONFIRM** 虚拟按键，修改被应用，页面退回主屏，串口输出：`Confirm button pressed. Threshold changes applied.`。
*   **旁白配音**：
    *   “最后展示我们基于触摸屏的图形化阈值设定。在主界面点击 SETTINGS 虚拟按键即可进入设置菜单。我们可以通过点击屏幕上的 UP 或 DOWN 虚拟触控键，以 1.0°C 为步长快速调整温度报警范围。点击 CONFIRM 即可直接应用，并安全回退至主屏幕，交互逻辑极其直观。”
*   **字幕贴图**：
    *   **拓展要求（2）图形化阈值设置交互**
    *   ✅ SETTINGS 面板：温度/气压上下限可视化调整
    *   ✅ 1.0°C 步长整步微调，点击 CONFIRM 瞬时应用
